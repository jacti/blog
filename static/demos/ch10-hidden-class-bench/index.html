Hidden Class vs Dictionary 모드 벤치마크

delete 한 번이 프로퍼티 접근 속도에 어떤 영향을 주는지 직접 확인해 보세요.

Fast 모드
const obj = {
  a: 1, b: 2,
  c: 3, d: 4, e: 5
};
// Hidden Class 유지됨
undefined 할당
const obj = {
  a: 1, b: 2,
  c: 3, d: 4, e: 5,
  temp: 0
};
obj.temp = undefined;
// Hidden Class 유지됨
delete 사용
const obj = {
  a: 1, b: 2,
  c: 3, d: 4, e: 5,
  temp: 0
};
delete obj.temp;
// Hidden Class 파괴!
