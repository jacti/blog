LLM 기반 챗봇을 웹 인터페이스로 구현한 프로토타입입니다.

## 시스템 구성

{{< tabs >}}
{{< tab name="프론트엔드" >}}
Next.js와 Tailwind CSS로 채팅 UI를 구성하고, 응답 스트리밍을 지원합니다.
{{< /tab >}}

{{< tab name="백엔드" >}}
Edge Function에서 프롬프트 가공과 토큰 제한을 처리합니다.
{{< /tab >}}

{{< tab name="감시" >}}
로그와 프롬프트/응답 샘플을 Supabase에 저장합니다.
{{< /tab >}}
{{< /tabs >}}

## 향후 개선 아이디어

- 음성 입력과 TTS 출력 추가
- 안전 가이드라인을 위한 금칙어 필터링
- 대화 QA 데이터셋으로 회귀 테스트 자동화
