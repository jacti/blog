## 머신러닝이란?

머신러닝은 데이터에서 패턴을 학습하여 예측하거나 분류하도록 컴퓨터를 훈련시키는 기법입니다.

{{< callout type="info" title="핵심 개념" >}}
머신러닝 모델은 **데이터**, **특징(feature)**, **학습 알고리즘**을 조합하여 성능을 높입니다.
{{< /callout >}}

### 지도 학습

라벨이 있는 데이터로 입력과 정답의 관계를 학습합니다.

- 회귀: 연속값 예측
- 분류: 카테고리 예측

### 비지도 학습

라벨이 없는 데이터를 통해 숨겨진 구조를 찾습니다.

- 클러스터링: 유사한 데이터 묶기
- 차원 축소: 정보를 유지한 채 데이터 압축

### 함께 보면 좋은 자료

{{< steps >}}
{{< step title="데이터 전처리" >}}
모델 학습 전 데이터를 정리하고 이상치를 제거합니다.
{{< /step >}}

{{< step title="모델 선택" >}}
문제 유형에 맞춰 선형 모델, 트리 기반 모델, 신경망 등을 선택합니다.
{{< /step >}}

{{< step title="평가와 개선" >}}
검증 데이터로 성능을 확인하고 하이퍼파라미터를 조정합니다.
{{< /step >}}
{{< /steps >}}
